# VibeCode Docs

Một bộ docs site được khởi tạo bằng `/vibe-scaffold docs`.

- 📚 MDX-first — viết Markdown, nhúng React component khi cần
- 🔎 Tìm kiếm sẵn — Nextra full-text search out of the box
- 🌐 Đa ngôn ngữ — `vi` + `en` trong `theme.config.tsx`
- ⚡ Vercel-ready — deploy bằng 1 click

Bắt đầu tại [Bắt đầu → Cài đặt](/intro/getting-started).
