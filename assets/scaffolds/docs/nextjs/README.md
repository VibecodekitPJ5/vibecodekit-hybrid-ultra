# Vibecode Docs scaffold (Pattern D — Documentation)

A Nextra-powered documentation site with sidebar + table-of-contents +
global search out of the box.  Generated by `/vibe-scaffold docs`.

## Local dev

```bash
npm install
npm run dev    # → http://localhost:3000
```

## Add a page

Drop an `.mdx` file under `pages/<section>/` and add an entry to
`pages/<section>/_meta.json` to position it in the sidebar.

## Search

Optional Algolia DocSearch — set the three `NEXT_PUBLIC_ALGOLIA_*` env
vars in `.env.local`, then `npm run build`.  Without them the local
search ships with codeblock indexing disabled (see `next.config.mjs`).

## Internationalisation

`vi` (Tiếng Việt) and `en` (English) are wired in `theme.config.tsx`.
Add a sibling file `<page>.<locale>.mdx` to localise.
