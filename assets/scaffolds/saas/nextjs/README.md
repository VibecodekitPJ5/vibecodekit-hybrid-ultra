# SaaS starter (VibecodeKit scaffold)

Generated by `vibecodekit.scaffold_engine` ‚Äî preset `saas`, stack `nextjs`.

Stack: **Next.js 15 (App Router)** + **NextAuth (Credentials stub)** + **Prisma (SQLite)** + Tailwind.

## Run

```bash
npm install
npx prisma generate
npm run dev
```

Visit:

- `/` ‚Äî public landing
- `/auth/login` ‚Äî login form (NextAuth Credentials stub)
- `/dashboard` ‚Äî protected dashboard placeholder
- `/api/auth/[...nextauth]` ‚Äî NextAuth handler

## Layout split (Pattern B)

| Route group  | Description |
|--------------|-------------|
| `app/`       | Public pages (landing, marketing copy) |
| `app/auth/`  | Login / register flow |
| `app/dashboard/` | Authenticated user area |

3. Add server-side session checks (`getServerSession`) on dashboard
